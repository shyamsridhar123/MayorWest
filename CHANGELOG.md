# Changelog

All notable changes to Mayor West Mode will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-15

### Features

- Interactive CLI setup wizard with full/quick/custom modes
- YOLO auto-approval for safe Copilot commands
- 4-layer security architecture (Actor Allowlist, Protected Paths, Kill Switch, Audit Trail)
- GitHub Actions workflows for autonomous task orchestration
- Auto-merge workflow for Copilot PRs
- Issue template for mayor-task creation
- Status command to check configuration
- Verify command to validate setup
- Examples command for task templates

### Security

- CODEOWNERS-based actor allowlist
- Protected paths configuration
- Kill switch for emergency stops
- Full audit trail via PR comments

### Documentation

- Comprehensive README with security-first approach
- CLI guide with usage examples
- Technical Requirements Document (TRD)
- Quick reference card
- Testing guide
